
# Simple Ad Blocker Chrome Extension

Simple Ad Blocker is a lightweight Chrome extension that blocks unwanted ads, providing you with a cleaner and faster browsing experience.

## How to Use

### Installation

1. Download or clone this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions`.
3. Enable "Developer mode" by toggling the switch in the upper-right corner.
4. Click on the "Load unpacked" button.
5. Select the directory where you cloned or downloaded this repository.

### Usage

Once installed, the extension will start blocking ads automatically. Simply browse the web as usual, and enjoy an ad-free experience!
